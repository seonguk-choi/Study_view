:: 코딩 컨벤션 (코드 컨벤션)
  ※ Naming Rule 에 대해서도~ 찾아보세요~!
  - PascalCase : 클래스 선언등 프로그래밍에서 사용
  - camelCase : html, css등에서 사용하고 그밖에 프로그래밍 언어에서 사용
  - kebab-case : html, css 등에서 사용하는 규칙
  - snake_case : python 언어에서 사용하는 기본 규칙



:: 파비콘, 패비콘

  - 북마크(=즐겨찾기)할때 사이트 이름 옆에 표기하는 작은 이미지
  - favorite + icon = favicon
  - 최소 16x16 ~ 512 x 512 : 디바이스에 따라 알아서 적용
  - *.ico / *.png : 단 두가지 포맷(format)만 사용 가능
  - 이미지 캡쳐, 다운 --> favicon generator 활용 !! -->자동으로 다양한 크기의 파비콘으로 변경 --> 다운로드 및 적용!!
